# Wet as F\*\*k - Multi-Source Video Aggregator

A modern, feature-rich video streaming platform that aggregates content from multiple sources into a single, elegant interface.

## Current Features

- 🎬 Multi-source video aggregation (7mmtv, eporner, hqporner, javmost, missav, motherless, pornhoarder, supjav, sxyprn, vjav, xnxx)
- 🔍 Powerful search functionality
- ❤️ Favorites/Bookmarks system
- 📚 Watch history tracking
- 📋 Playlists and library management
- 🔥 Trending, recent, and popular video feeds
- 👤 Artist/Performer profiles
- 🌙 Dark/Light theme support
- ⚙️ User settings and preferences
- ⚡ Modern tech stack (Next.js 16, React 19, Tailwind CSS)
- 📊 Vercel Analytics & Speed Insights integration

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Tech Stack

- **Framework**: Next.js 16
- **UI Library**: React 19
- **Styling**: Tailwind CSS 4
- **Component Library**: shadcn/ui
- **State Management**: Zustand
- **Video Player**: Plyr
- **Icons**: Tabler Icons
- **Animation**: Motion
- **Analytics**: Vercel Analytics & Speed Insights
- **Theme**: next-themes
